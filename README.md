# Aria2c-dart



## Features
1. Connect to an aria2 jsonrpc server, control the aria2 downloader remotely.
2. Support http, websocket.
3. Support secret authentication


## Getting started

```
flutter pub add aria2
```

## Usage
You can use this package like this:

```dart
const like = 'sample';
```
